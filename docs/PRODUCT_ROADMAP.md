
# PRODUCT_ROADMAP.md — Logist.kg

## Phase 1 — MVP

Duration: 3–4 weeks

Features:
- User registration
- Authentication
- User roles
- Create freight
- Freight search
- Driver bids
- Basic profiles

Goal:
Working logistics marketplace.

---

## Phase 2 — Platform Core

Duration: 4–6 weeks

Features:
- Chat system
- Reviews & ratings
- Notifications
- Vehicle management
- Company profiles
- Dispatcher tools

Goal:
Complete logistics workflow.

## Phase 3 — Monetization

Duration: 3–5 weeks

Features:
- Paid freight listings
- Subscriptions
- Advertisements
- Promoted loads

Goal:
Generate revenue.

Build user trust.

- Analytics dashboard

- Infrastructure scaling
